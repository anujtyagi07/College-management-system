This is a full-stack College Management System built using the **MERN stack** (MongoDB, Express.js, React.js, Node.js).  
The application provides a centralized platform to manage academic and administrative operations for colleges, including students, faculty, courses, notices, and timetables with role-based access.

---

## Features

### Admin Features
- Manage faculty accounts and profiles  
- Manage student records with enrollment and academic details  
- Create and manage academic branches  
- Manage subjects and courses by semester and branch  
- Create and publish notices  
- Upload and manage timetables  
- Profile and password management  

### Faculty Features
- View and update personal profile  
- Upload and manage study materials (notes, assignments, syllabus)  
- Organize materials by subject, semester, and type  
- Upload timetables  
- Search students by enrollment number, name, or semester  
- View notices and announcements  
- Password reset and profile management  

### Student Features
- View personal and academic details  
- Access study materials by subject and type  
- View and download timetables  
- View notices and announcements  
- Update profile and manage password  

---

## Tech Stack
- Frontend: React.js  
- Backend: Node.js, Express.js  
- Database: MongoDB  
- Authentication: JWT
